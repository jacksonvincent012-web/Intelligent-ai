🌐 How to connect

Backend mode — Run V-Mind locally (python run_server.py), enter the URL.

API Key mode — Enter any OpenAI/OpenRouter key and chat directly from the browser. No backend needed.

Your key is stored locally and never sent anywhere except the API provider.
